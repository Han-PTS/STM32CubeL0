Update History


V1.11.0 / 26-October-2018

Main changes

  • Removed support of Atollic TrueSTUDIO toolchain.
  • system_stm32l0xx.c file
    • Updated system_stm32l0xx.c files to correct comments for VECT_TAB_OFFSET definition.
    • Removed call to __HAL_RCC_PWR_CLK_DISABLE() in SystemClock_Config() to limit power consumption.
    • Updated default MSI_VALUE reset value set in SystemCoreClock.
    • Updated SystemCoreClockUpdate() function to check HSI16DIVF for HSI divided by 4.
  • stm32l0xx_hal_conf.h
    • Updated default MSI_VALUE reset value set in SystemCoreClock.
    • Added USE_SPI_CRC definition to be aligned with SPI driver.
  • RTC/RTC_Tamper
    • Updated RTC/RTC_Tamper example to reset Tamper status after writing to backup register.
  • PWR/PWR_OptimizedRunMode
    • Rename LL_PWR_IsActiveFlag_VOSF() to LL_PWR_IsActiveFlag_VOS() to be compliant with all STM32 series.
  • Renamed HAL_I2C_Slave_Sequential_Receive_IT to HAL_I2C_Slave_Seq_Receive_IT to be aligned with I2C driver
    • I2C/I2C_TwoBoards_RestartAdvComIT
    • I2C/I2C_TwoBoards_RestartComIT
  • Renamed HAL_TIM_SlaveConfigSynchronization to HAL_TIM_SlaveConfigSynchro to be aligned with TIM driver
    • TIM/TIM_PWMInput

V1.8.0 / 25-November-2016

Main Changes

  • Updated stm32l0xx_hal_conf.h files with latest version from STM32L0xx_HAL_Driver.
  • Renamed RCC_PLLMUL_x to RCC_PLL_MULX and RCC_PLLDIV_X to RCC_PLL_DIVXto be aligned with others series.
  • Added new projects I2C to introduce I2C repeated start feature:
    • I2C_TwoBoards_RestartAdvComIT
    • I2C_TwoBoards_RestartComIT
  • Updated I2C examples after HAL I2C driver update:
    • I2C_TwoBoards_AdvComIT
    • I2C_TwoBoards_ComDMA
    • I2C_WakeUpFromStop
  • Changed to use USART output:
    • I2C/I2C_TwoBoards_RestartAdvComIT
  • Updated sampling times naming in ADC examples:
    • ADC/ADC_LowPower
    • ADC/ADC_AnalogWatchdog
  • Replaced HAL_COMP_Start_IT() by HAL_COMP_Start() in Comp_Config():
    • COMP/COMP_AnalogWatchdog
  • Updated/added new projects IWWDG/WWDG aligned with updated HAL IWDG/WWDG drivers:
    • IWDG/IWDG_Reset (replacing IWDG/IWDG_Example project)
    • IWDG/IWDG_WindowMode
    • WWDG/WWDG_Example

V1.7.0 / 31-May-2016

  • No changes compared to previous version (V1.6.0).
